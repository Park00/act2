# 리액트 프로젝트 환경 구축하기

## 현재 내 환

```text
This package will install:
	•	Node.js v14.15.3 to /usr/local/bin/node
	•	npm v6.14.9 to /usr/local/bin/npm
```

## Getting local environment

```
$ npm install --no-package-lock
$ npm run dev
```

> 여기까지 설치하기까지 갖은 고생을 했는데 남겨 놓은게 없어서 아쉬움
